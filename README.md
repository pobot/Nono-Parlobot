Nono-Parlobot
=============

Sources codes pour Nono le robot parlant.

Le projet est pédagogique : vous trouverez bon nombre de petits programmes de test, pas seulement le programme des démonstrations. Nono étant en exemplaire unique, il évolue régulièrement donc on aura un sous-répertoire pour chaque événement temporel, car on ne peut maintenir un seul code source (les capteurs et les cartes électroniques changent).
